# Training Logs

This directory contains log files generated by training scripts.

## Log File Naming Convention

```
{script_name}_{model}_{timestamp}.log
```

Examples:
- `train_00_plain_text_medgemma-4b_20251226_100000.log`
- `train_01_medical_dict_medgemma-4b_20251226_110000.log`
- `train_02_kor_med_test_medgemma-27b_20251226_120000.log`

## Progress Files

Progress files (`progress_*.txt`) provide quick status updates:
- Created when training starts
- Updated at each step/epoch
- Useful for monitoring long-running jobs

## Usage in Scripts

```python
from logging_utils import TrainingLogger

# Create logger
tlog = TrainingLogger("train_01_medical_dict", model="medgemma-4b")

# Log messages
tlog.info("Loading data...")
tlog.step(100, loss=0.5, accuracy=75.0)
tlog.epoch(1, val_loss=0.4)
tlog.validation(kormedmcqa_accuracy=78.5)
tlog.finish(final_accuracy=85.0)
```

## Viewing Logs

```bash
# Latest log for a script
tail -f log/train_01_medical_dict_medgemma-4b_*.log

# Progress file
watch -n 5 cat log/progress_train_01_medical_dict_medgemma-4b.txt

# List all logs
ls -lt log/*.log | head -10
```

## Log Cleanup

Old logs can be removed manually or with:
```bash
# Remove logs older than 7 days
find log -name "*.log" -mtime +7 -delete
```
